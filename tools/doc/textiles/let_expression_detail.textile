h2. module

h3. "index":index.html > "ES Next":es_next_detail.html > let expression

let式です。スコープを生成します。

*実装済み処理系* : Mozilla1.7以降

*基本的な例*

<notextile>
<pre class="brush: js">
var foo = 200;
let (foo = 300) {
  console.assert(foo === 300);
}
</pre>
</notextile>

*変換後*

<notextile>
<pre class="brush: js">
var foo = 200;
!function(foo) {
  console.assert(foo === 300);
}(300);
</pre>
</notextile>

スコープを生成します。let式の入り口に定義した値はlet式内のスコープでのみ有効になります。
