h2. ネイティブモジュール

*module 'fs'*
　
*class Dir*

インスタンスメソッド

|_. メンバ|_. インターフェース|_. 説明|
|Dir|Dir(dirname)|コンストラクタです。引数で指定したディレクトリに対する操作を行います。|
|entries|entries(recursive)|コンストラクタで指定したフォルダ内のエントリの数だけEntryクラスのインスタンスを配列で返します。|
　
スタティックメソッド
　
|_. メンバ|_. インターフェース|_. 説明|
|mkdir|mkdir(dirname)|dirnameで指定されたディレクトリを生成します。
|rm|rm(name)|nameで指定されたファイル、ディレクトリを削除します。|
|chdir|chdir(dirname)|dirnameで指定されたディレクトリにカレントディレクトリを変更します。|
　
　
*class Entry*
　
インスタンスメソッド
　
|_. メンバ|_. インターフェース|_. 説明|
|fullpath|fullpath()|エントリの絶対パスを返します。|
|name|filename()|エントリの名前のみを返します。|
|dir|dir()|エントリの親ディレクトリを返します。|
